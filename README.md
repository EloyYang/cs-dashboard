# 고객센터 인입 대시보드

구글시트 로우데이터(seed, seed(정산용), 조치사항, 실시간원격주치의, 컴퓨터주치의, remote, 현황)를
실시간으로 불러와 분석하는 단일 HTML 대시보드입니다.

## 사용 방법

- **웹으로 접속 (권장)**: GitHub Pages 주소로 접속하면 PC·맥북·모바일 어디서든 바로 사용할 수 있습니다.
- **PC에 내려받아 사용**: 이 저장소를 ZIP으로 내려받아 압축을 풀고, 로컬 웹서버로 열어야 합니다
  (브라우저 보안 정책 때문에 `index.html`을 더블클릭으로 열면 구글시트 데이터를 가져올 수 없습니다).
  - Windows: 폴더에서 `python -m http.server 8899` 실행 후 `http://localhost:8899` 접속
  - Mac: 터미널에서 폴더로 이동 후 `python3 -m http.server 8899` 실행 → `http://localhost:8899`

## 구성

| 파일/폴더 | 설명 |
|---|---|
| `index.html` | 대시보드 본체 (메인 · 인입 분석 · 점검자 분석 · 검색 · 보고서) |
| `대시보드 템플릿/` | 보고서 내보내기용 엑셀 양식 3종 — 양식이 바뀌면 이 파일을 교체 후 다시 배포 |

## 보고 양식(템플릿) 교체

`대시보드 템플릿/` 안의 파일을 최신 보고 파일로 교체한 뒤 커밋·푸시하면 됩니다.
파일명은 그대로 유지해야 합니다: `기술실적관리.xlsx`, `LGU+_원격점검현황.xlsx`, `LGU+소호_원격점검현황.xlsx`

## 대시보드 수정 후 재배포

바탕화면의 `고객센터 대시보드.html`을 수정한 뒤, 이 저장소 폴더의 `배포.bat`을 실행하면
최신 파일이 `index.html`로 복사되고 자동으로 커밋·푸시됩니다.
